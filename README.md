# Bookbot

BookBot is my first [Boot.dev](https://www.boot.dev) project. This counts how many words and characters are in a text file.

## How To Use

```sh
python main.py <path-to-text-file>
```

I assume you have already installed the Python programming language on your machine.

## Example

```sh
python main.py books/frankenstein.txt
```
